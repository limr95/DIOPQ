# DIOPQ
药品验证管理系统（DIOPQ）是一款专为制药企业设计的Windows桌面应用程序，用于药品确认与验证的台账管理。
本系统采用纯前端技术，数据存储在本地，不联网，不上传，确保数据安全与隐私。

## 解决痛点
- ✅ 手工记录易出错、难追溯
- ✅ Excel管理格式不统一
- ✅ 过期提醒不及时
- ✅ 多人协作数据不同步
- ✅ 审计追踪不完整

## 核心功能
### 1. 智能台账管理
- **全流程记录**：涵盖方案时间、报告时间、验证周期、下次验证时间等完整信息
- **多维度分类**：支持质量部、各车间、公用系统等多类别管理
- **灵活分类体系**：分类选项根据实际数据自动生成，适配不同企业需求

### 2. 过期预警系统
- **三色预警**：
    - 🔴 已过期 - 红色背景高亮，包括"过期"、"合格"、"待验"、"进行"四种状态中已过期的项目
    - 🟡 本月到期 - 黄色背景提醒，包括"待验"、"合格"、"进行"三种状态中本月到期的项目
    - 🔵 近2-3月到期 - 蓝色背景预警，包括"待验"、"合格"、"进行"三种状态中近2-3月到期的项目
- **看板统计**：顶部实时显示各到期状态数量，一键筛选查看

### 3. 数据导入导出
- **多格式支持**：Excel (.xlsx, .xls)、JSON、CSV
- **灵活导入模式**：
    - 删除原数据导入
    - 增量导入（追加）
    - 全量导入（去重合并）
- **样式导出**：支持带样式的Excel导出，保持表格格式

### 4. 审计追踪
- **全程记录**：新增、修改、删除、导入、导出、打印等操作
- **详细信息**：操作时间、用户、类型、对象、详情、IP地址
- **筛选查询**：按操作类型、时间范围快速查找
- **导出打印**：支持审计追踪记录导出PDF或打印

### 5. 批量操作
- **批量删除**：勾选多条记录一键删除
- **智能筛选**：按分类、状态、日期范围、关键词多条件筛选
- **快速清除**：一键清除所有筛选条件

### 6. 本地数据存储
- **安全可靠**：数据保存在本地应用目录，无需联网
- **保存机制**：增加、删除、导入等操作自动保存，也可以手动导入、导出
